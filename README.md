# Clinical Decision Support and Pattern Recognition in Healthcare

## Overview

This repository contains my submission for the Cotiviti Intern Assessment.

The project explores the use of Artificial Intelligence (AI) in clinical decision making and pattern recognition within healthcare. It includes:

- A research report discussing current trends, opportunities, challenges, and strategic recommendations.
- A Proof of Concept (POC) demonstrating AI-assisted clinical decision support.
- A PowerPoint presentation summarizing the report and prototype.
- A recorded demonstration of the project.
